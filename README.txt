## 这是一个未完成的项目
在复习SSM的过程中，参考《Spring MVC+Mybatis企业级应用实战》书籍中的内容所作。但因其没有提供前端页面，而我志又不在前端，所以没有完全完成，保存该练习代码仅供以后实际操作参考。
## 项目问题
1. get和post请求没有分开，比如添加操作，修改操作，可以改用get操作获取添加界面，而处理post请求完成实际操作
2. 项目较为简单，导致Service层和Dao层功能几乎一致，可能这是这本书主要讲解的是Spring MVC和Mybatis的内容，基本没有涉及AOP等内容
3. 该项目的前端界面为我从github找一个类似的项目中复制，项目前端内容仍有可借鉴的地方，比如iframe的使用，可以在使用中有一个较好的体验。
4. 该项目中分页功能已提供，但前端界面中并没有使用，前端和后端不太匹配，加上是一个帮助我理解书上内容的项目，就没有深入完善。
5. Mybatis常用xml格式进行映射，该项目采用的是注解映射配合动态sql，整个部分太耦合，在实操过程中可以选择xml格式，这样代码和操作数据库的sql语句是分开的，更方便维护。
